About Me

Hi there! My name is Ishani. I'm a rising junior studying Computer Science with a minor in Machine Learning at Carnegie Mellon University. I live in Cupertino, CA but I'm originally from Pune, India.

At CMU, I have taken a wide variety of courses on topics such as Theoretical Computer Science, Data Structures and Algorithms, Computer Systems, Type Theory and Probability and Statistics. As a research assistant, I worked on Nomos, a programming language for smart contracts. I added functional constructs to the language, developed a theory for its subtyping and implemented this algorithm in the typechecker. I also served as an EXCEL Leader during the 2019-2020 school year. As an EXCEL Leader for classes on Discrete Math and 3D-Calc, I conducted weekly sessions of 6-10 students, prepared handouts for the sessions, and implemented various collaborative learning techniques in my sessions. Projects

  1. Nomos - CMU CS Department

    Nomos is a programming language for smart contracts based on session types. As a part of this project, I:

    • Wrote a parser, type checker and interpreter for the functional layer of Nomos.
    • Integrated this typechecker with that for the session-typed layer of Nomos to build a bidirectional typechecker for the combined language.
    • Developed the theory and implementation for the subtyping of Nomos.
    • Implemented several smart contracts such as a puzzle, a wallet and a bank account in Nomos.

  2. Summer 2019 Internship Project - IBM Research

    In this project, I leveraged Go, Kubernetes and Shell Scripts to:

    • Build a Kubernetes operator that handles resource management, queueing and dispatching of Apache Spark jobs in a multi-cluster environment, with an emphasis on making the operator code as generic as possible.
    • Develop a Kubernetes "Failfast" operator that allows users to configure policies for marking jobs as failed and ready for redispatch.
    • Investigate and carry out performance testing on the Google Spark-on-k8s-Operator.

  3. Study Buddy - Tartan Hacks 2019

    In this project, we:

    • Developed a SaaS application that scanned handwritten or typed notes, converted them into text and highlighted important phrases to allow readers to quickly extract key information from the document.
    • Utilized Bootstrap to automatically create and display flashcards of key concepts from the text.

    The application was built in Python and deployed on Microsoft Azure.
